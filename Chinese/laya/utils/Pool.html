Packagelaya.utils
Classpublic class Pool
InheritancePool Object

Pool 是对象池类，用于对象的存储、重复使用。

合理使用对象池，可以有效减少对象创建的开销，避免频繁的垃圾回收，从而优化游戏流畅度。
