load()方法 
public static function load(url:String, type:String = null):Deferred

函数在此处阻塞，等待指定地址的资源加载完成后，唤醒函数继续向下执行。

参数

url:String — 资源地址。
 
type:String (default = null) — 资源类型。

Returns
Deferred — 处理当前阻塞的一个 Deferred 对象。
notify()方法 
public static function notify(conditions:* = null, value:* = null):void

发送消息。

参数

conditions:* (default = null) — 消息条件。若值为 null，则默认传入当前语句所在的函数。
 
value:* (default = null) — 消息传递的数据。

sleep()方法 
public static function sleep(delay:int):void

函数在休眠一定时间后，继续向下执行。

参数

delay:int — 休眠时间，单位是毫秒。

wait()方法 
public static function wait(conditions:*):*

函数在此处阻塞，等待条件消息发布后，即当 notify(conditions,value) 执行且 conditions 相等时，唤醒函数继续向下执行。 其中 wait 的返回值为 notify 的参数 value 的值。

参数

conditions:* — 唤醒条件。

Returns
* — 唤醒时被传递的数据。
