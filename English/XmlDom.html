nodeNameproperty 
public var nodeName:String

The name of the node that specifies the node. If the node is an element node, the nodeName property returns the label name. If the node is an attribute node, the nodeName property returns the name of the property. For other node types, the nodeName property returns different names of different node types.

nodeTypeproperty 
public var nodeType:int

Returns the node type of the specified node with the numeric value. If the node is an element node, the nodeType property will return 1. If the node is an attribute node, the nodeType property will return 2.
