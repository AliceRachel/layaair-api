Classpublic class SoundManager
InheritanceSoundManager Object

SoundManager It is a sound management class. It provides a playback control method for background music and sound effects. The engine acquiescence has two sets of sound schemes: WebAudio and H5Audio. Play sound effects, give priority to WebAudio play sound, if WebAudio is not available, it is played with H5Audio, and H5Audio has compatibility problems on some machines (such as no mixing, playback delay, etc.). Play background music, use H5Audio to play (using WebAudio will increase the extra large memory, and wait until after loading to play, there is delay). It is suggested that the background music should be of MP3 type and wav or MP3 type for sound effects (if it is packaged as app, the sound effect can only be in wav format). For detailed tutorials and sound formats, please refer to: http://ldc.layabox.com/doc/?nav=ch-as-1-7-0

playMusic()method 
public static function playMusic(url:String, loops:int = 0, complete:Handler = null, startTime:Number = 0):SoundChannel

Play background music. The background music can only be played at the same time. If you call this method again when the background music is played, the background music will be stopped first and the current background music will be broadcast.

Parameters

url:String — The address of the sound file.
 
loops:int (default = 0) — The number of cycles, 0 is an infinite loop.
 
complete:Handler (default = null) — Sound play completes the callback.
 
startTime:Number (default = 0) — The start time of the sound play.

Returns
SoundChannel — SoundChannel object, through this object can control the voice, and get voice information.

setSoundVolume()method 
public static function setSoundVolume(volume:Number, url:String = null):void

Set the sound volume. Depending on the parameters, you can set the volume of the specified voice (background music or sound effect) or all sound effects (excluding background music).

Parameters

volume:Number — Volume. The initial value is 1. Volume range from 0 (mute) to 1 (maximum volume).
 
url:String (default = null) — (default = null) sound playback address. The default is null. For empty, set the volume of all sound effects (excluding background music) and do not empty to indicate the volume of the specified sound (background music or sound effect).
