Classpublic class Graphics
InheritanceGraphics Object

Graphics The class is used to create a drawing display object. Graphics can draw multiple bitmaps or vector graphs at the same time, and can also change the effect of drawing with instructions such as save, restore, transform, scale, rotate, translate, alpha and so on. Graphics is stored in command flow and can access all command streams through CMDS properties. Graphics is a lighter object than Sprite, and reasonable use can improve application performance (such as a Graphics command set that transforms a large number of node drawings to a node, reducing the consumption of a large number of nodes).
