Packagelaya.display
Classpublic class Input
InheritanceInput Text Sprite Node EventDispatcher Object

Input Class is used to create display objects to display and enter text.

Input Classes encapsulate native text input boxes, which result in a little deviation between the location of the default text of the object and the location of the text when the user clicks on the input due to the difference between the different browsers.

Property Detail
asPasswordproperty
asPassword:Boolean

This API has been abandoned. Use type= "password" instead of setting asPassword, asPassword will be deleted in the next major update.

Specifies whether a text field is a password text field.

If the value of this property is true, the text field is regarded as a cryptographic text field, and the input characters are hidden by using asterisk instead of actual characters. If false, the text field will not be regarded as a cryptographic text field.

The default value is false.
