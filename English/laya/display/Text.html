Text A class is used to create a display object to display the text.

Note: if the runtime system does not find the set font, it will render the text with the default font of the system, resulting in an exception. (usually the computer is normal, and some mobile terminals display an exception due to the lack of the font to be set).

    lang()method 
    public function lang(text:String, arg1:* = null, arg2:* = null, arg3:* = null, arg4:* = null, arg5:* = null, arg6:* = null, arg7:* = null, arg8:* = null, arg9:* = null, arg10:* = null):void

    Extracts the text content of the current language from the language pack according to the specified text. And the {i} text in this text is replaced.

    After setting up the Text.langPacks language package, you can use Lang to get the language inside.

    For example:

  • (1) the value of text is "my name" and is taken to the value "My name" in the current language version corresponding to the text, and the "My name" is set as the content of the current text.
  • (2) the value of text is "congratulations on winning {0} diamonds, {1} experience." The value of arg1 is 100, and the value of arg2 is 200. Then we get the value of "Congratulations on your winning {0} diamonds, {1} experience." in the current version of the text corresponding to the text. Then the {0} and {1} in the text are replaced by the values of arg1 and arg2 starting from 0. Set the processed text "Congratulations on your winning 100 diamonds, 200 experience." to the content of the current text.
