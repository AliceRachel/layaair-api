Classpublic class AnimationPlayerBase
InheritanceAnimationPlayerBase Sprite Node EventDispatcher Object

The animation playback base class provides the basic animation playback control method and the frame tag event related function.

You can inherit this class, but do not instantiate the class directly, because some methods need to be implemented by subclasses.

intervalproperty 
interval:int

Frame interval time (unit: millisecond) for animation. The default value depends on Config.animationInterval=50, and the default frame interval can be modified by Config.animationInterval.

To set independent frame intervals for an animation, you can use set interval. Note that if the animation is playing, it will reset the starting time of the frame loop timer for the current time, that is, if the interval is set frequently, the time interval of the animated frame update will be slower than the expected, or even not. To update。

play()method 
public function play(start:* = 0, loop:Boolean = true, name:String, showWarn:Boolean = true):void

Start playing animation. The play (...) method is designed to be called at any time after the instance is created, and when the corresponding resource is loaded, the animated frame filling method (set frames) is invoked or the instance is displayed on the stage, it will judge whether it is playing, if so, it is played.

With the wrapMode attribute, the animation play sequence type can be set.

Parameters

start:* (default = 0) — (optional) specify the index (int) or the String of the start of the animation. The frame labels can be added and deleted by addLabel (...) and removeLabel (...).
 
loop:Boolean (default = true) — (optional) whether to circulate.
 
name:String — (optional) animation name.
 
showWarn:Boolean (default = true)
