Packagelaya.display
Classpublic class Stage
InheritanceStage Sprite Node EventDispatcher Object

Stage It's a stage class that displays the root node of the list, and all the displayed objects are displayed on the stage. Through the Laya.stage single case.

Stage provides a number of adaptation modes. Different adaption patterns will produce different canvas sizes. The larger the canvas, the greater the pressure of the rendering, so the appropriate adaptation options are selected.

Stage provides different frame rate modes, the higher the frame rate, the greater the rendering pressure, the more electricity, the reasonable use of the frame rate or even the dynamic change of frame rate, which is beneficial to improving the power consumption of the mobile phone.
