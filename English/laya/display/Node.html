InheritanceNode EventDispatcher Object
Subclasses Sprite

Node Class is the base class that can be placed in all objects in the display list. The display list manages all objects displayed in the Laya runtime. Use Node The class arranges the display objects in the display list. The Node object can have a child display object.

destroy()method 
public function destroy(destroyChild:Boolean = true):void

Destroy this object. The destroy object will remove itself from the parent node by default, and clean up its own reference relationship, waiting for the JS automatic garbage collection mechanism to be recovered. Destroy can not be used again.

Destroy removes its own event listener, its own timer listener, removes child objects and removes itself from the parent node.

Parameters

destroyChild:Boolean (default = true) — (optional) if the child node is destroyed at the same time, if the value is true, the child node will be destroyed, otherwise the child nodes will not be destroyed.

public function timerLoop(delay:int, caller:*, method:Function, args:Array = null, coverBefore:Boolean = true, jumpFrame:Boolean = false):void

A function is executed repeatedly. The function is the same as Laya.timer.timerLoop ().

Parameters

delay:int — Interval time (per unit of milliseconds).
 
caller:* — The execution domain (this).
 
method:Function — The callback method at the end of the time.
 
args:Array (default = null) — (optional) callback parameters.
 
coverBefore:Boolean (default = true) — (optional) whether the delay execution is covered by default is true.
 
jumpFrame:Boolean (default = false) — Whether the clock is skipping. A time based loop callback in which a number of callbacks can be performed in a unit time interval. For performance considerations, the engine executes only once by default, and after setting the jumpFrame=true, the callback will be executed several times in a row.
