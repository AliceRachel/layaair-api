destroy()method
override public function destroy(destroyChild:Boolean = true):void

Destroy this object. The destroy object will remove itself from the parent node by default, and clean up its own reference relationship, waiting for the JS automatic garbage collection mechanism to be recovered. Destroy can not be used again.

Destroy removes its own event listener, its own timer listener, removes child objects and removes itself from the parent node.

Parameters

destroyChild:Boolean (default = true) — (optional) if the child node is destroyed at the same time, if the value is true, the child node will be destroyed, otherwise the child nodes will not be destroyed.
