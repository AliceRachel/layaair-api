It encapsulates location, width and height processing for UI components.

sizeGridproperty 
sizeGrid:Array

The current instance is effectively scaling the grid data.

If set to null, the entire display object is normally scaled when any scaling conversion is applied.

Data format: [upper edge, right margin, bottom margin, left margin, repeat filling (value 0: not repeated filling, 1: repeated filling)].

  • For example: [4,4,4,4,1]

sizeGrid

  1. top margin
  2. Right distance
  3. margin-bottom
  4. left
  5. Repeat filling (value 0: no duplicate filling, 1: repeated filling)

When defined sizeGrid Property, the display object is split to sizeGrid

  • The area above the rectangle
  • The upper right corner outside the rectangle
  • The area on the left side of the rectangle
  • The right side of the rectangle
  • The lower left corner outside the rectangle
  • The region below the rectangle
  • The right lower corner outside the rectangle
  • The upper left corner outside the rectangle
At the same time, it also supports the 3 palaces, such as the 0,4,0,4,1 is the horizontal 3 palace, 4,0,4,0,1 is the vertical 3 palace, and the 3 palace is higher than the 9 palace.
