keyLocationproperty 
keyLocation:uint  [read-only]

The position of the key on the keyboard. This is very useful for distinguishing keys that appear on the keyboard many times. For example, you can distinguish between the left Shift key and the right Shift key based on the value of this property: the value of the left Shift key is KeyLocation.LEFT, and the right Shift key is KeyLocation.RIGHT. Another example is the distinction between the standard keyboard (KeyLocation.STANDARD) and the numeric keys pressed on the numeric keypad (KeyLocation.NUM_PAD).
