Classpublic class TileTexSet
InheritanceTileTexSet Object

This is a sub texture class, including the management of similar animations. TiledMap divides the texture into numerous sub textures, and also subsets one of the sub textures into an animation sequence. The implementation of this class is that if the sub texture is found to be replaced by an animation sequence, animationKey will be set to true. That is, animationKey is true, TileAniSprite is used to display, and the animation sequence is drawn to TileAniSprite on time.
