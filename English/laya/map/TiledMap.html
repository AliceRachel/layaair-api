Packagelaya.map
Classpublic class TiledMap
InheritanceTiledMap Object

TiledMap is the core of the whole map Maps are used to map maps at different levels (e.g., surface, vegetation, and construction). Each layer uses GridSprite to handle the display objects only in the viewport area. Each block includes a N lattice (tile) Lattice types are divided into two types: animation lattice and picture lattice.
