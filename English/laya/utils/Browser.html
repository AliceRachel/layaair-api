clientHeightproperty 
clientHeight:Number  [read-only]

The browser window can be seen as the height. Obtained by analyzing the browser information. The browser's multiple attribute value priorities are: window.innerHeight (including scrollbar height) > document.body.clientHeight (no scrollbar height) > document.documentElement.clientHeight (no scrollbar height), and if the former is 0 or empty, the latter is selected.


Implementation
    public static function get clientHeight():Number
clientWidthproperty 
clientWidth:Number  [read-only]

The browser window can be seen as a width. Obtained by analyzing the browser information. The browser's multiple attribute value priorities are: window.innerWidth (including the scroll bar width) > document.body.clientWidth (excluding the scroll bar width), and if the former is 0 or empty, the latter is selected.
