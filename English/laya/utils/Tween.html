from()method 
public function from(target:*, props:Object, duration:int, ease:Function = null, complete:Handler = null, delay:int = 0, coverBefore:Boolean = false):Tween

From the props property, move to the current state.

Parameters

target:* — Target object (which is about to change the object of the attribute value).
 
props:Object — List of changed attributes, such as {x:100, y:20, ease:Ease.backOut, complete:Handler.create (this, onComplete), update:new Handler (this, onComplete)}.
 
duration:int — The time spent, in milliseconds.
 
ease:Function (default = null) — The slow motion type, by default, is uniform motion.
 
complete:Handler (default = null) — End the callback function.
 
delay:int (default = 0) — Delay execution time.
 
coverBefore:Boolean (default = false) — Whether to cover the previous delay.

Returns
Tween — Return to the Tween object.
