Repeated execution.

Parameters

caller:* — The execution domain (this).
 
method:Function — The timer callback function.
 
args:Array (default = null) — Callback parameters.
 
coverBefore:Boolean (default = true) — Whether the delay execution before coverage is default is true.
 
jumpFrame:Boolean (default = false) — Whether the clock is skipping. A time based loop callback in which a number of callbacks can be performed in a unit time interval. For performance considerations, the engine executes only once by default, and after setting the jumpFrame=true, the callback will be executed several times in a row.
