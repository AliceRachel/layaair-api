Handler Is an event processor class.

It is recommended that Handler.create () method be created from object pool to reduce object creation consumption. Created Handler After the object is no longer used, you can use Handler.recover () to recycle it to the object pool, and do not use this object again after recycling, otherwise it will lead to unforeseen errors.

Be carefulï¼š Because the mouse event also uses this object pool, incorrect recovery and calling may affect the execution of mouse events.
