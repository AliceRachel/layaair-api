radix:int (default = 0) — The cardinal number of the number to be parsed. The default value is 0, indicating 10 hexadecimal, and the other values are between 2 and 36. If it starts with "0x" or "0X", it will base on 16. If this parameter is not within the above range, this method returns 0.
