clearRes()method 
public static function clearRes(url:String, forceDispose:Boolean = false):void

Clean up the cache for the specified resource address. If it is Texture, it is destroyed by reference counting. [note] if the picture itself is in the automatic collection (the default picture is less than 51212), the memory can not be destroyed, and this picture will be managed by the aggregate manager of the large graph.

Parameters

url:String — Resource address.
 
forceDispose:Boolean (default = false) — If the destruction is mandatory, some resources are destroyed by reference counting. If forceDispose=true, the reference count is ignored and directly destroyed, such as Texture, and the default is false

clearTextureRes()method 
public static function clearTextureRes(url:String):void

Destroy the image resources used by Texture, retain the texture shell, and automatically recover if the image resources used by texture are not available when rendering next time. Compared to clearRes, clearTextureRes just clean up the picture resources used in the texture, do not destroy the texture, and automatically restore the picture resource when it is used again. And clearRes will destroy texture thoroughly, causing no more use; clearTextureRes ensures immediate destruction of picture resources without worrying about destroying errors, and clearRes is destroyed by reference counting. [note] if the picture itself is in the automatic collection (the default picture is less than 51212), the memory can not be destroyed, and this picture is managed by a large collection Manager.

Parameters

url:String — Atlas address or texture address, such as Loader.clearTextureRes ("res/atlas/comp.atlas"); Loader.clearTextureRes ("hall/bg.jpg");

load()method 
public function load(url:String, type:String = null, cache:Boolean = true, group:String = null, ignoreCache:Boolean = false):void

Load the resources. Loading errors will send Event.ERROR events with incorrect parameters.

Parameters

url:String — Resource address.
 
type:String (default = null) — (default = null) resource type. Optional values are Loader.TEXT, Loader.JSON, Loader.XML, Loader.BUFFER, Loader.IMAGE, Loader.SOUND, Loader.ATLAS, Loader.FONT. If NULL, the type is analyzed according to the file suffix.
 
cache:Boolean (default = true) — (default = true) whether data is cached.
 
group:String (default = null) — (default = null) group name.
 
ignoreCache:Boolean (default = false) — (default = false) ignore the cache and force the reload.
