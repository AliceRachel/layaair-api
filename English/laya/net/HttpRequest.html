HttpRequest By encapsulation of HTML XMLHttpRequest The object provides full access to the HTTP protocol, including the ability to make POST and HEAD requests, as well as common GET requests. HttpRequest It only provides an asynchronous form to return the response of the Web server, and can return content in text or binary form.

Be carefulï¼š It is suggested that a new one be used for each request HttpRequest Objects, because each time the send method is called, the data is emptied and the state of the HTTP request is reset, which leads to the resetting of the request that has not returned before the response, so that the response results of the previous request are not obtained.

send()method
public function send(url:String, data:* = null, method:String = get, responseType:String = text, headers:Array = null):void

Send a HTTP request.

Parameters

url:String — The address of the request. Most browsers implement a homogenous security policy and require that the URL have the same host name and port as the script containing the text.
 
data:* (default = null) — (default = null) data sent.
 
method:String (default = get) — (default = "get") the HTTP method used for the request. The values include "get", "post", "head".
 
responseType:String (default = text) — (default = "text") the response type of the Web server can be set to "text", "JSON", "XML" and "arraybuffer".
 
headers:Array (default = null) — The header information of the (default = null) HTTP request. The parameter is like an array of key-value: key is the name of the head and should not include blank, colon or line wrap; value is the value of the head and should not include line wrap. such as ["Content-Type", "application/json"] .
