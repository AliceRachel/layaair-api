Media is used to capture cameras and microphones. You can capture either one or both. getCamera Can be used before supported() Check whether the current browser is supported. NOTE:

At present, Media only supports Android on mobile platforms, and does not support IOS. It can only be used completely in FireFox, and video cannot be captured during Chrome testing.
